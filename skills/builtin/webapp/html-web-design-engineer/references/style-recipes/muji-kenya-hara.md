# Recipe: muji-kenya-hara

Adapted from the Garden web-design-engineer recipe family for Mindar single-file HTML web apps.

## Character

Quiet, humble, precise, and material. Use for minimal tools, reflective summaries, simple calculators, and focused interfaces where restraint is the point.

## Layout

- Prefer simple vertical rhythm, clear grouping, and a small number of views.
- Make every element earn its place.
- Use labels, whitespace, and alignment rather than decorative containers.
- Keep the first viewport calm but useful.

## Visual Tokens

- Background: off-white, light gray, or soft paper neutral.
- Accent: sparse, muted, used for action and selection only.
- Borders: subtle 1px rules.
- Radius: small to medium.
- Shadows: almost none.

## Typography

Use system sans with careful sizing and line-height. Avoid display type unless the content truly needs it.

## Interaction

- Good fits: simple segmented controls, toggles, input forms, calculators, and collapsible details.
- States should be clear without visual noise.

## Avoid

- Maximal dashboards.
- Loud gradients.
- Over-decoration that undermines calm.
