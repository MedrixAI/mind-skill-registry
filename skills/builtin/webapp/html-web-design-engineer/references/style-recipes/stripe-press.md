# Recipe: stripe-press

Adapted from the Garden web-design-engineer recipe family for Mindar single-file HTML web apps.

## Character

Clear, thoughtful, bookish technology publishing. Use for learning modules, docs-like explainers, guided courses, concept walkthroughs, and structured knowledge products.

## Layout

- Build a guided path: overview, concepts, examples, checks, and next steps.
- Use a readable main column with side navigation, progress, and compact interactive checks.
- Use diagrams and code-like examples when they clarify the source material.
- Keep controls visually calm and consistent.

## Visual Tokens

- Background: warm off-white or pale neutral.
- Accent: one refined product/editorial color.
- Surfaces: clean panels with subtle borders, not heavy shadows.
- Radius: 6-12px.
- Motion: gentle reveal for learning progression only.

## Typography

Use system sans for UI and system serif for occasional editorial headings if helpful. No external fonts.

## Interaction

- Include concept tabs, example toggles, progress steps, knowledge checks, flashcard-like reveals, or glossary search.
- Provide clear selected and completed states.

## Avoid

- Overly playful classroom visuals.
- Dense dashboard styling for learning flows.
- Fake quizzes with fabricated answers not grounded in sources.
