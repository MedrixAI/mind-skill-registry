# Recipe: mailchimp-freddie

Adapted from the Garden web-design-engineer recipe family for Mindar single-file HTML web apps.

## Character

Friendly, confident, and approachable. Use for playful-but-professional apps, onboarding, decision helpers, and creative learning tools.

## Layout

- Use clear task paths, friendly helper text, and obvious controls.
- Pair approachable copy with disciplined spacing and strong accessibility.
- Use illustration-like inline SVG or CSS shapes only when they explain or guide.
- Keep professional polish; playful does not mean childish.

## Visual Tokens

- Background: light neutral with one warm accent family.
- Accent: saturated but controlled.
- Surfaces: simple panels, readable contrast, clear action hierarchy.
- Radius: moderate, 10-16px.
- Motion: small feedback transitions; avoid distracting loops.

## Typography

Use system sans with friendly weights and generous line-height. No external fonts.

## Interaction

- Good fits: guided steps, sliders, toggles, cards with reveal states, friendly empty states, and scenario calculators.
- Every playful cue must also be usable with keyboard and focus.

## Avoid

- Emoji-as-icon defaults.
- Candy-colored palettes without hierarchy.
- Fake personality content not present in sources.
